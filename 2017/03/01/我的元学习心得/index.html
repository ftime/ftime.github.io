我的元学习心得

Posted on 2017-03-01   |     |  

网上各种付费课程很多，花了金钱时间精力却没有学到东西的话，就非常可惜。为了避免这种情况，除了要认真筛选课程之外，我觉得在学习其他东西之前，首先学习一下如何学习，提高自己的学习能力变得越来越重要。xdite（郑伊廷）老师把这个学习过程叫做“元学习”，我前段时间学习了这个课程，觉得挺好。

在武侠小说中，只有内功深厚的人才能拥有高超的武功，如果没有内功作为基础，练习各种武功，特别是各种招式是没有实际意义的。如果一项技能就是一门武功的话，“元学习”就是提高内功的方法。

作为兴趣，我最近在学习吉他，写作和游泳。当我反思自己正在学习也乐在其中的这几样东西时，我发现在学习的过程中有些方法竟然无意间与元学习的理论相互吻合，包括以下三点：高频小套路，成就感，及时反馈。

高频小套路

高频小套路指的就是最核心，最常出现，可以重复练习的基础技能，而高级技能都是这些基础技能的组合。

拿学吉他来说，我跟的是许岑的指弹吉他教程，从零开始，不懂乐理知识，没有节奏感，但是抓起吉他就练了。许岑在课程中专门教了一个左手按弦练习的高频小套路，他把左手四个指头按弦顺序的所有可能组合拿出来，编成了一个套路谱“1324 3412 4231 2143 1342 3124”。1是食指，2是中指，3是无名指，4是小指。

怎么练习呢？开始可以从第一根弦第一品格开始练习，左手四个指头都用上，右手只用食指和中指。1324指的是，依次左手食指按第1品，右手食指拨弦；左手无名指按第3品，右手中指拨弦；左手中指按第2品，右手食指拨弦；左手小指按第4品，右手中指拨弦。依次类推，可以从任意一根弦的任意一品格开始，甚至每拨4次就换一根弦，但是左手的指法顺序不变，一直练习到非常熟练。许岑说他曾经一边看港剧《大时代》一边一直练习这个基本功，反复边看边练好几遍。我刚开始完全不理解这个练习的用处，但是只管生硬地练习了。当我在学习第一首曲子的过程中，我逐渐领会到了这个基本功的作用，因为那个套路会不时地出现在曲子中，之前的练习很大地降低了学习的困难度。

成就感

我在15年12月申请了微信公众号，但是并没有怎么写。一直到16年8月份才开始一周更新一篇，虽然读者不多，但是成就感不断。

最初不好意思，根本不敢在朋友圈里分享自己的文章，但是竟然有陌生人关注我，这可能是一个随机事件，但是仍然给我带来了不少喜悦。后来我开始在朋友圈里分享自己的文章，文章的阅读量从几个人增长到几十个人再到上百人，这给了我不少成就感。

由于还没有获得原创功能，不能获得读者的评论，但是看到自己的文章被陌生人转发，看到订阅的读者在增加，心里还是很开心的，这对我来说，就是一种积极的反馈。至于有些文章被别人收藏，那对我来说就是随机奖励。

除了微信公众号，我还把把文章发表在简书上，看到自己的写作字数在一点点地增加，偶尔有篇文章被首页投稿收录，这些都在不断给我注入新的动力。

写作过程中，这些切实可见的“奖励”，让我一路写了过来。

及时反馈

我常常在周末和老婆一起去游泳，这对我俩来说不只是一项运动，更是一种休息，一种游戏。我们都非常享受那蓝色的静谧。

我最初是看视频学的蛙泳，花了很长时间才学会，而且划水效率很低，50米的泳道，我要划水40次，又累又慢。游泳池里不能录像，所以我根本不知道自己的姿势哪里不对，以为自己的身姿到位了，但是实际上可能差得很远。自己没时间上游泳班，就让老婆参加了三期游泳课，她高级班毕业后，我跟她一起游，她在旁边看我的泳姿，及时地对我的错误进行矫正，结果很快我的划水效率就大大提高，50米的泳道，划水25次就能游完，而且能一直游下去。练习了三四个小时之后，原来游50米就大喘气的我，竟然也能一口气游600米。

以上是我自己学习过程中的一点点体会，这些都是积极的方面。然而，要想要把元学习课的内容全部应用起来，并不是一件容易的事情。

比如学吉他，我买的是网络视频课程，因此根本无法获得老师的及时反馈，也就是说，必须要靠我的观察能力，和老师的进行对比，才可能发现错误之处并进行纠正，而这个过程是极其费时费力的，成就感的获取也非常困难。因为进步通常都是缓慢的，这个时候就要依赖于自己感受细微变化的能力，要想尽办法让自己感受到正在发生在自己身上的变化，而我用的方法就是录音和录像，时不时地录音保存一下，过一段时间找出一个十多天前的录音听一听，而后再录一遍比较一下，这样就能更明显地感受到那一点点的进步，体会到成就感…

距离课程结束已经两个多月了，至今依然印象深刻的是“大牛并不是坚持，而是对自己的事业上瘾”。我想到了论语里的那句话，“知之者不如好之者，好之者不如乐之者。”，与君共勉。

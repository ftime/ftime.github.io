韩国大选在即，总统给我打电话了

Posted on 2017-05-07   |     |  


在韩国工作生活多年，我对韩国政治并不关心，但是最近总统大选太热闹，各种宣传信息满天飞，躲都躲不掉。

本来在公园里跑步，远处马路上，某党的宣传队在用高音喇叭播放宣传歌，魔性之强以至于宣传车已经走了，那旋律和名字仍在脑中单曲循环。老婆干脆继续跟着哼唱，搞得我哭笑不得。

这种大街上洗脑式的宣传并不稀奇，当年朴槿惠参加选举的时候就经历过一次。不过，这次选举出了新花样—电话和短信宣传。

前些天下午，我接到一个陌生电话，接通后那边说：你好，我是文在寅。我当时一惊：总统候选人啊，给我打电话干啥？没等我开口，那边巴拉巴拉继续说，我才意识到这是一个拉票的骚扰电话，挺新鲜，索性听下去，电话内容大意是，为了大韩民国的未来，你投票选我吧，不必等到5月9号最终大选日才能去投票，4号5号就能投，赶紧的…

在韩国，接到各种录音骚扰电话的频率很高，不过大多是推销手机的，推销总统还是头一回。

候选人是有编号的，1号候选人文在寅进行电话宣传之后，其他的候选人也不甘示弱：这样拉票也可以？！于是2号洪準杓也赶紧发短信拉票：

大意是说：“选我吧，我要把烟的价格降下来、把燃油价格降一半、电话费降40%、保留司法考试，赦免信用不良的人、赦免为了生计而犯罪的人。”这个候选人除了在马路上的魔性歌曲里，在别的地方没听说过，不过看了拉票短信我就觉得不靠谱。价格应该是市场说了算，而是否犯罪应该是法律说了算，怎么能由一个总统说了算呢？

3号安哲秀也发来短信。这位之前听说过，是一位不断转行的学霸。最初是在韩语课本上学习过他的故事，从医学博士转行做杀毒软件（安博士软件的创始人），后来当大学教授，再进入政界。如此赫赫有名，但发来的短信倒是不入眼。

大意是：“选择前两位就是选择回到老旧的过去；选我就是选择未来。我会成为一个不说谎的总统，不让国民羞愧的总统。…”（的确，很多韩国同事都觉得朴槿惠太给韩国人丢脸了）。

“不说谎的总统”，看到这样的话，我只想说：安学霸没怎么读过心理学啊，呵呵…有心理学调查显示，普通人平均一天要说谎100次之多，谎言是生活的一部分，更是政治的一部分。选举宣传时说这样的话，不是搬起石头砸自己的脚嘛！我应该说他足够勇敢吗？

和老婆聊这个，她淡淡的说了一句：他们都是历史舞台上的小丑而已。PS：她最近看《人类简史》正入迷，跟我聊的都是智人打败了其他人种，人类的融合，帝国，宗教，资本主义，科学革命的话题。在这样的视角下，电视上候选人跳出来互相揭短表现作秀，不是小丑，又是什么呢？

PS：虽然总统候补有15位之多，但是选票和焦点主要集中在这前3位身上，以现在的形势来看，总统应该就会在1号文在寅和3号安哲秀之间产生。政府网站上还公布了候选人的财产情况：

转行之王安哲秀果然不一般，他的赚钱能力甩出对手几十条街。安哲秀有1196亿韩元的资产，约合人民币7.4亿，而对手文在寅的资产只有18亿韩元，约合人民币1千万。

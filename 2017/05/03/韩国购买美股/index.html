投资：在韩国购买美股

2017-05-03
  • 投资

我对股票市场一无所知，但是在读了李笑来的《财富自由之路》专栏之后，决定拿出收入的一小部分购买GAFATA的股票，也就是google，apple，facebook，amazon，tencent，alibaba的股票，采用定投的方式进行。具体来说就是每月定时定额（收入的10%）购买上述公司的股票，而不管当时价格的高低。

做了决定，也获得了老婆的同意，三个月之后，我终于开始行动了。人在韩国，所以步骤可能跟国内不太一样。趁着公司的一个三明治强制休假日，着手进行准备工作。

第一步：到券商开户。

我选择的是现代证券，因为一个有多年炒股经验的朋友选择的这家，所以我也选择了这家。毕竟是外国人，有诸多不便，别人走过这条路，至少说明能走通，没什么坑，有问题也能够相互交流。

到了证券公司才发现，公司改名了，叫KB证券。告诉柜台人员，我要开户，购买海外的股票。柜员似乎是第一次给外国人办理开户业务，花了很长时间询问周围的同事，打电话咨询，等等。确认可以开户之后，开始办理，除了我的登陆证和护照之外，这个柜员还要我提供中国纳税号码，我说没有，但柜员开户要求提供国籍所在国的纳税号码是全世界的规则。我查询了一下，看到有人说身份证号码就是纳税号码。于是提交了身份证，她扫描留底，算是通过。添了四五张表格之后，开户成功。对方给我一个有账号的卡和一个密码卡。然后帮我在手机的券商APP global able 上进行账号的设置，设置ID，设置认证书等等，最终测试可以正常登陆后，算是开户成功，整个过程耗时将近3个小时。

第二步：到银行开外汇账户。

上一步开通之后，就可以把韩元转账到该账户，然后在APP上进行换汇，购买海外的股票了。但是考虑到有时需要用人民币投资，那就需要把人民币兑换成美元汇到韩国，这时就需要在韩国有一个接受美元外汇的账户。

于是，我到友利银行开通了外汇账户。开通时又费了不少时间，因为办事员又是第一次办理这种业务，同样的在给我开通之前问上级，打电话确认账户的类型。经过种种确认之后，告诉我外国人可以开通的只能是一个受限制比较多的账户。所有的操作都必须到营业厅进行，每年向海外汇款的自由额度是5万美金，当然有收入证明的话，可以汇出收入额度的资金。

第三步：到证券公司开通外汇关联账户

第一步在证券公司开通的主账户只能接受韩元，要想直接存入美元的话，就需要让券商给自己开通一个能接受美元的关联账户。这样在收到美元外汇之后，就可以直接拿来投资，而不要美元换成韩元，韩元再换成美元的多次汇率损失。

这个美元的关联账户开通起来很容易，把要求告诉柜员之后，柜员让我选择KEB外换银行或者KB银行的虚拟账户。在选择之后，就生成了一个虚拟账户，这是一个没有存折的虚拟账户。当把美元打到这个账户之后，美元会自动转到证券公司的主账号之下，直接进行投资。

其他注意事项：

股票单次交易的手续费：美股，交易额的0.25%，最低手续费5.5 USD。也就是说2000USD以下都是5.5USD的手续费。
中国银行海外汇款的手续费：汇款额的0.1%（最低50 CNY）+150 CNY电汇费。也就是说5万人民币以下都是200CNY的手续费。

最基本的投资硬件已经具备，可以购买海外股票了；而最关键的投资的智慧，则需要不断地学习，提高。
